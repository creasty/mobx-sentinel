Staticget

  • get<T extends object>(subject: T, formKey?: symbol): Form<T>

  • Get the form instance for a subject.

    • Returns the existing form instance if the subject is already associated with one.
      Otherwise, creates a new form instance and associates it with the subject.
    • The form instance is cached in the internal registry, and it will be garbage collected when the subject is no longer in use.
      In rare cases, you may need to manually dispose the form instance using Form.dispose.

    Type Parameters

    • T extends object

    Parameters

    • subject: T

      The subject to associate with the form

    • OptionalformKey: symbol

      The key to associate with the form. If you need to associate multiple forms with the same subject, use different keys.

    Returns Form<T>

    Throws

    TypeError when the subject is not an object.
